# instrumental-variables-simulation

Implementation of an instrumental variables (2SLS) framework, including analysis of the first-stage condition and exclusion restriction assumptions.

Monte Carlo simulations comparing ordinary least squares and instrumental variable estimators in terms of bias, variance, and mean squared error under varying sample sizes, instrument strengths, and degrees of endogeneity, supported by theoretical derivations of bias and consistency.

For results, see notebook/analysis.ipynb
